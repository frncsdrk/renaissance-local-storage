# renaissance-local-storage

local storage adapter for renaissance

## Usage

- include `localStorage.js` in your paths
- load it in your page component
- call the include

## API

.save({string} key, {string} value)  
.update({string} key, {string} value)  
.get({string} key)  
.remove({string} key)  
.clear()
